# 🔎 Lost and Found Management System  
### Smart Item Recovery & Matching Application  
Built using Zoho Creator

---

## 📖 Project Overview

The Lost and Found Management System is a low-code web application developed using Zoho Creator to streamline the reporting, tracking, and matching of lost and found items within institutions such as colleges, offices, and public places.

The system allows users to report lost items, register found items, match them efficiently, and track verification status through a centralized dashboard.

---

## 🚨 Problem Statement

In many colleges and public places:

- Lost items are not tracked systematically
- No centralized database for found items
- Manual verification causes delays
- No transparency in item matching process

This application provides a structured, digital solution to improve recovery efficiency and reduce item loss confusion.

---

## 🎯 Objectives

- Digitally record lost and found items
- Enable item matching with status tracking
- Provide verification workflow
- Offer dashboard analytics
- Implement role-based management
- Improve item recovery efficiency

## 👥 User Roles

- Admin – Full access to manage matches, users, and verification
- Staff/User – Report lost/found items and view status

---

## 🔄 Workflow

1. User reports lost item
2. Found item is registered
3. Admin matches items
4. Verification is performed
5. Status updated (Confirmed / Pending / Rejected)
6. Item returned to rightful owner
